# quarkus-demo

## run by maven: 
1. mvn quarkus:dev
1. http://localhost:8080/swagger-ui/index.html?url=http://localhost:8080/api/swagger.json

## run by java
1. execute Main.java
1. http://localhost:8081/swagger-ui/index.html?url=http://localhost:8081/api/swagger.json
